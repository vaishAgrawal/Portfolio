## 📂 Getting Started

Clone the repository:

```bash
git clone https://github.com/vaishAgrawal/Portfolio.git
```

Navigate to the project directory:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

Open your browser and visit:

```text
http://localhost:3000
```
